# 작업 주의 사항
* conda confluence 환경을 사용할것
* html을 생성할때는 밝은 톤을 사용할것
* confluence_token.txt에 있는 confluence 토큰을 사용할 것
* 슬랙에 접근할 때는 slack_token.txt를 사용할 것 
* 지금 작업하는 confluence_summary 폴더의 상위 폴더에서 작업하는 것과 파일 삭제, 컨플루언스 글 삭제, 컨플루언스 글 내용 변경 등의 문제가 아니면 언제나 니가 자동으로 진행해줄것

# history.html 작성 지시
### 대상 내용
* 항상 나의 지시 원본 그대로를 글상자 형태
* 시간을 함께 기록
* 나의 지시를 해석한 너의 해석
* 너의 작업 진행 과정과 결과
* 만약 니가 작업을 진행하다가 에러를 만나면, 그 에러는 꼭 기록하고, 니가 해결한 방법을 꼭 함께 기록하도록해.
